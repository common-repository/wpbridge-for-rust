Integrates Wordpress sites with Rust servers to show player statistics and server information.

== Description ==

WPBridge for Rust connects your Rust server to WordPress, enabling you to show near real time server and player statistics directly on your WordPress site using shortcodes.

== Features ==

* Connects your Rust server to WordPress.
* Easy server statistics through shortcodes.
* Show the number of kills, headshots, suicides, loot statistics, server information and much more.
* rust-servers.info integration. 
* rustmaps.com integration. 

https://www.youtube.com/watch?v=iVYoRrPm1qo&list=PLQ6Ojd6qy2eME4l0zEA-UmvbxXTkJOsm7&index=1&t=2s

== Installation ==

1. Log in to your WordPress backend and navigate to _Plugins â†’ Add New.
2. Type "WPBridge" into the Search and hit Enter.
3. Locate the WPBridge plugin in the list of search results and click Install Now.
4. Once installed, click the Activate link.

== Configuration ==

1. Locate and click the WPBridge for Rust menu item in the sidebar menu.
2. Click Generate to generate your unique secret and then click Save Settings.
